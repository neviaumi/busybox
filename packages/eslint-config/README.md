# @busybox/eslint-config

## Installation

- Install dependencies

```bash
npm install --save-dev @busybox/eslint-config
```

- Setup [prettier](../prettier-config)

- In your project root `eslint.js`

```js
'use strict';

module.exports = {
  extends: ['@busybox'],
};
```

[Example](../../.eslintrc.js)
